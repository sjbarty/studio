# **App Name**: OptiPic

## Core Features:

- Image Upload & Display: Allows users to upload an image file and displays it within the editor interface.
- Intuitive Crop Tool: Provides a resizable cropping area for users to precisely select and adjust the desired portion of their image.
- One-Click Compression Presets: Offers 'Low', 'Medium', and 'High' compression options to quickly reduce image file size based on common quality preferences.
- Custom Compression Settings: Enables users to define a specific target image size (e.g., in KB/MB) or a quality percentage for precise file size control.
- Real-time Preview: Displays a preview of the image as cropping and compression settings are adjusted, along with estimated file size reduction.
- Download Processed Image: Allows users to download the cropped and/or compressed image to their device.

## Style Guidelines:

- The chosen light color scheme emphasizes clarity and professionalism, suitable for precise image editing. The primary color, a muted blue (#4C89B2), provides a stable and modern foundation, reflecting the app's functional purpose without being distracting. It's desaturated to avoid overwhelming the image content.
- The background color, an extremely light blue-grey (#EBF0F3), maintains visual continuity with the primary hue while ensuring maximum contrast for image visibility and UI elements, contributing to a clean and airy feel.
- An analogous, brighter accent color, a clean cyan (#4CDDDD), is used for interactive elements like buttons and selection outlines, drawing attention without clashing with the overall subdued aesthetic.
- The 'Inter' sans-serif font is recommended for all text. Its modern, objective, and highly readable design aligns perfectly with the app's focus on functionality and precision in a minimalist interface.
- Utilize simple, outline-based icons for tools and actions (e.g., a scissors icon for crop, a compress icon) to maintain a clean and uncluttered visual experience.
- A clean, functional layout prioritizing the image preview area. Controls are positioned logically, often grouped by function, ensuring ease of access without obscuring the content being edited.
- Subtle, fluid transitions and feedback animations for actions like applying a crop or switching compression settings, providing a responsive and polished user experience without being intrusive.
